Figma

Figma — кросс-платформенный онлайн-сервис для дизайнеров интерфейсов и веб-разработчиков. Разработка интерфейсов происходит в онлайн-приложении

MySQL

MySQL — это система управления реляционными базами данных с открытым исходным кодом с моделью клиент-сервер.

HTML&CSS

HTML — это язык разметки гипертекста. Он позволяет создавать и структурировать разделы, параграфы, заголовки, ссылки и блоки для веб-страниц и приложений. CSS — это язык, который отвечает за визуальное представление документов пользователю.

SASS

Sass — это расширение CSS, которое придаёт мощи и элегантности этому простому языку. Sass даёт возможность использовать переменные, вложенные правила, миксины, инлайновые импорты и многое другое, всё с полностью совместимым с CSS синтаксисом.

Bootstrap

Bootstrap — это открытый и бесплатный HTML, CSS и JS фреймворк, который используется для быстрой вёрстки адаптивных дизайнов сайтов и веб-приложений.

JS&jQuery

JavaScript — язык сценариев, с его помощью можно создавать интерактивные html-документы, производить вычисления, выполнять проверку допустимости данных без обращения к серверу jQuery — библиотека JavaScript, содержащая в себе готовые функции языка JavaScript

PHP

PHP — это препроцессор гипертекста (HTML).
PHP — это серверный язык программирования.
PHP — это скриптовый, интерпретируемый язык

GIT

Git — это система управления версиями
